Why saliency can be misleading

A saliency map is usually read as evidence that the model used the highlighted region. That reading is too strong. Gradients and class activation maps expose local sensitivity, not necessarily causal dependence. The project therefore compares visual explanations against perturbation and concept-level evidence.

Grad-CAM

Grad-CAM weights convolutional feature maps by the gradient of the target class score. It is spatially smooth and easy to read, but its resolution is limited by the last convolutional feature grid.

alpha_k^c = (1 / Z) * sum_i sum_j d y^c / d A_ij^k L_GradCAM^c = ReLU(sum_k alpha_k^c A^k)

Integrated Gradients

Integrated Gradients accumulates input gradients along a path from a baseline to the image. In this project the baseline is a heavily blurred version of the image, not a black image, so average color and brightness are preserved.

IG_i(x) = (x_i - x'_i) * integral_0^1 dF(x' + a(x - x')) / dx_i da

Stress-test metrics

The experiment compares original and perturbed saliency maps with top-pixel IoU and Spearman rank correlation. Low overlap means the explanation changed even when the animal pixels were intended to remain semantically stable.

IoU = |S_original intersect S_perturbed| / |S_original union S_perturbed|

Concept-level methods

TCAV moves from pixels to semantic directions. A Concept Activation Vector is trained to separate examples that express a concept from examples that do not, then the class score is differentiated along that concept direction.

TCAV(c, k) = fraction_x [ directional_derivative f_k(x) along v_c > 0 ]

The background stress test

The key experimental move is to perturb the background while keeping the animal region as stable as possible. If the prediction or the explanation changes strongly, the model is not only reading morphology. It is also sensitive to context, texture and dataset correlations.

Interpretation

In the available sample, background swap changes the predicted class in 75% of evaluated cases and causes the strongest confidence drop. That is not enough to claim a universal property of AwA2, but it is strong evidence that the current classifier and its explanations are context-sensitive.

From pixels to concepts

Pixel saliency answers "where is the model sensitive?" Concept analysis asks a higher-level question: "which semantic attributes move the decision?" This is the right direction for the project because AwA2 includes class-level attributes such as horns, hooves, stripes, flippers and habitat-related concepts.

Phase 6 class-concept heatmap. This exposes the semantic prior encoded in AwA2 attributes before asking whether the neural model follows the same concepts.
Concept transition analysis. The antelope-to-walrus error is not a small semantic move: the concept cosine is 0.348, with large gains in water-related concepts and losses in horns, hooves and plains concepts.

TCAV results

Five concepts were tested at ResNet layer3: stripes, furry, hooves, horns and flippers. Every CAV reached training accuracy 1.0 on its selected positive/negative examples. That confirms linear separability in the sampled activation space, but also warns that the CAVs may capture broad correlated directions, not pure human concepts.

Phase 8: forcing the decision through concepts

TCAV is still post-hoc: it probes a model after it has learned its own representation. The Concept Bottleneck Model changes the architecture itself. The image is first mapped to 20 AwA2 semantic attributes, and the final classifier receives only those predicted concepts.

Interpretation

Phase 8 is an important positive result, but not a magic fix. The bottleneck makes the decision pathway more readable and only loses about 5.2 accuracy points relative to the direct baseline. At the same time, the predicted concepts can be noisy and some intervention effects are counterintuitive. This supports a careful conclusion: concept-based models improve auditability, but they also require concept-quality validation.

Interpretation

Phase 9 strengthens the conclusion without overselling it. The randomized-model comparison passes the basic sanity check because similarity is low. However, the modest SmoothGrad and Occlusion agreement shows that the exact saliency pattern is fragile and method-dependent. This is the final message of the project: saliency maps can be useful diagnostic objects, but visual plausibility is not enough evidence of faithful explanation.
